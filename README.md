## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn test:unit
yarn test:unit:watch
yarn test:snapshot:
yarn test:snapshot:watch
yarn test:cypress:watch
yarn cypress:run
yarn cypress:open
```
